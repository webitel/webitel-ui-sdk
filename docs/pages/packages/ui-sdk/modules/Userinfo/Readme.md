## `v1` vs `v2`?

### `v1`

> [!WARNING]
> **Deprecation warning!** `v1` – застарілий. По можливості, використовуйте `v2`.

* старенький, 
* працює (майже) всюди (`v25.08`), 
* але має певні проблеми та обмеження:

  1. **Кодова база – заплутана** (при чому, не в хорошому сенсі), та важка для розуміння.
  Не вистачає якоїсь структуризації коду, і виходить що він [вермішельний](https://en.wikipedia.org/wiki/Spaghetti_code).  

  2. **Немає можливості перевірки по ліцензії на конкретну сутність**.


### `v2`

* **новий**
* значно більш **структурований**
* значно **гнучкіший**
* ... але все одно складний (станом на `v25.08`).

### Який обрати?

**Краще v2**. 

**Але!** якщо вам треба код, копі-пастнутий з сусіднього розділу, то ймовірно
що темплейтний код + v1 буде оптимальніше.
