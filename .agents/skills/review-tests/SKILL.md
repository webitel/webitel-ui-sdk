# Review existing tests

## Description

Review existing tests, correct them and provide feedback on their quality and coverage.

## Instructions

1. Review existing tests in this project
2. Automatically fix exisiting tests if they are not correct
3. If some tests are valid, but the source code is not, approve source code changes first before fixing source code issues
4. Delete tests that are not needed, outdated or incorrect
5. Delete tests that are badly written, not readable or not maintainable
6. Provide feedback on the quality and coverage of the tests
7. Return the corrected tests

## Notes

1. Do not use automated scripts to fix tests, always review the tests manually
